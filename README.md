# Secretário de Suporte

O Secretário de Suporte é uma ferramenta de linha de comando (CLI) baseada em Python que atua como um assistente de IA focado em tarefas de suporte técnico e redação profissional. Ele se integra nativamente a um repositório local do Obsidian utilizando técnicas de Geração Aumentada por Recuperação (RAG), permitindo buscar contexto de documentos de arquitetura, manuais e histórico de tickets para gerar respostas precisas e auxiliar no preenchimento de resoluções de forma automatizada com a API do Gemini.
